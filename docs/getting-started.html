Getting Started with FORGE

Install FORGE, set up your API keys, and run your first AI-powered command in 5 minutes.

Prerequisites

  • Node.js 18 or higher
  • npm or yarn package manager
  • DeepSeek API key (free)
  • GitHub token (optional, for GitHub features)
1

Install FORGE globally

npm install -g @2tinteractive/forge-cli

Or if you prefer yarn:

yarn global add @2tinteractive/forge-cli
üí° Tip: You can also use npx for one-off commands without installation: 
npx @2tinteractive/forge-cli ask "Hello"
2

Get your DeepSeek API key

1. Visit DeepSeek Platform

2. Sign up or log in

3. Create a new API key

4. Copy the key (starts with sk-)

3

Set up your API key

Option A: Environment variable (recommended for security)

export DEEPSEEK_API_KEY="your-key-here"

Option B: Use FORGE config (stored securely in keychain)

forge config set deepseek.api_key your-key-here
üîí Security: Your API key is never shared and stays on your machine.
4

Verify installation

forge --version

You should see: FORGE version 1.0.0 

forge ask "Hello, what can you help me with?"

You should see a streaming response from DeepSeek AI!
