API Reference

Complete API documentation for FORGE's programmatic interface.

Core API

initialize(config?: ForgeConfig): Promise<ForgeInstance>
forge.initialize({ apiKey: 'sk-xxx', cwd: '/project' })

Initialize FORGE with configuration options.

Parameter Type Description
config.apiKey string DeepSeek API key (optional, can use env)
config.cwd string Working directory (default: process.cwd())
config.theme string UI theme: 'glass' | 'dark' | 'light'
Returns: ForgeInstance with all methods
ask(prompt: string, options?: AskOptions): Promise<string>
await forge.ask('How do I fix this bug?', { context: fileContent })

Send a prompt to DeepSeek AI.

Parameter Type Description
prompt string Your question or prompt
options.context string | string[] File contents or context to include
options.model string AI model to use
options.stream boolean Stream response (default: true)

File System API

fs.readFile(path: string): Promise<FileContent>
const content = await forge.fs.readFile('src/index.js')

Read a file and return its content with metadata.

Returns: { content: string, path: string, size: number, modified: Date }
fs.writeFile(path: string, content: string, options?: WriteOptions): Promise<void>
await forge.fs.writeFile('test.txt', 'Hello World', { force: true })
Parameter Type Description
options.force boolean Overwrite existing file
options.createDirs boolean Create parent directories
fs.listDir(path?: string, options?: ListOptions): Promise<FileInfo[]>
const files = await forge.fs.listDir('./src', { recursive: true })
Option Type Description
recursive boolean List recursively
includeHidden boolean Include dotfiles
filter RegExp Filter by pattern
fs.search(pattern: string, path?: string): Promise<string[]>
const jsFiles = await forge.fs.search('*.js', 'src/')

Search for files matching pattern.

GitHub API

github.auth(token: string): Promise<UserInfo>
const user = await forge.github.auth('ghp_xxx')

Authenticate with GitHub.

github.clone(repo: string, options?: CloneOptions): Promise<LocalRepo>
const repo = await forge.github.clone('owner/repo', { depth: 1 })
Option Type Description
branch string Branch to clone
depth number Shallow clone depth
path string Local path
github.createPR(options: PROptions): Promise<PR>
const pr = await forge.github.createPR({ title: 'Fix bug', head: 'fix', base: 'main' })
Parameter Type Description
title string PR title
body string PR description
head string Head branch
base string Base branch
draft boolean Create as draft

Database API

db.connect(config: DBConfig): Promise<Connection>
const conn = await forge.db.connect({ host: 'localhost', user: 'root', database: 'forge' })
db.query(sql: string, options?: QueryOptions): Promise<QueryResult>
const result = await forge.db.query('SELECT * FROM users', { optimize: true })
Option Type Description
optimize boolean AI-optimize query
timeout number Query timeout (ms)
Returns: { rows: any[], fields: any[], time: number }
db.optimizeQuery(sql: string): Promise<OptimizedQuery>
const optimized = await forge.db.optimizeQuery('SELECT * FROM large_table')
Returns: { query: string, explanation: string, improvement: number }

PHP API

php.run(code: string, options?: PHPRunOptions): Promise<PHPResult>
const result = await forge.php.run('echo "Hello";', { version: '8.2' })
Option Type Description
version string PHP version (7.4-8.3)
memoryLimit string Memory limit (e.g., '256M')
timeout number Execution timeout (s)
php.generatePlugin(name: string, description: string, features?: string[]): Promise<Plugin>
const plugin = await forge.php.generatePlugin('Contact Form', 'AJAX contact form', ['spam-protection'])

Testing API

test.generate(code: string, options?: TestOptions): Promise<string>
const tests = await forge.test.generate(userCode, { framework: 'jest', coverage: 80 })
test.run(testPath: string): Promise<TestResults>
const results = await forge.test.run('./tests')
Returns: { passed: number, failed: number, skipped: number, coverage: number }
test.debug(file: string, breakpoints?: number[]): Promise<DebugSession>
const session = await forge.test.debug('script.js', [10, 25])

Start interactive debugging session.

Self-optimize component performance.

Error Handling

try/catch with typed errors
try {
  await forge.github.createPR({...});
} catch (error) {
  if (error instanceof ForgeAuthError) {
    console.log('Authentication failed:', error.message);
  } else if (error instanceof ForgeAPIError) {
    console.log('API error:', error.statusCode, error.body);
  } else if (error instanceof ForgeValidationError) {
    console.log('Invalid input:', error.field, error.reason);
  }
}
Error Types
  • ForgeAuthError - Authentication issues
  • ForgeAPIError - API call failures
  • ForgeValidationError - Invalid input
  • ForgeTimeoutError - Operation timed out
  • ForgeFileError - File system errors
