    AI-Native Methods

    Every FORGE component exposes these AI-native methods:

    explainState()

    Returns current component state for AI consumption. Includes props, context, and performance metrics.

    getPossibleActions()

    Returns all possible AI actions this component supports, with parameters and descriptions.

    optimize()

    Self-optimizes component performance based on usage patterns.

    suggestImprovements()

    AI suggests code improvements for the component.

    Example: GlassButton Manifest

    {
  "component": "GlassButton",
  "state": {
    "isEnabled": true,
    "variant": "primary",
    "size": "md",
    "isLoading": false
  },
  "possibleActions": [
    {"action": "click", "description": "Simulate button click"},
    {"action": "disable", "description": "Disable the button"},
    {"action": "setVariant", "params": ["primary|secondary|danger"]}
  ]
