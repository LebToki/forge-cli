ğŸ“Š Optimize Slow MySQL Query
Take a slow query and let FORGE optimize it with AI, then compare performance.
âŒ Before (3.2 seconds) 
SELECT * FROM posts 
LEFT JOIN comments ON posts.id = comments.post_id 
LEFT JOIN users ON comments.user_id = users.id 
LEFT JOIN post_meta ON posts.id = post_meta.post_id 
WHERE posts.status = 'published' 
AND posts.created_at > '2024-01-01'
âœ… After (0.8 seconds) 
SELECT 
    p.id, p.title, p.content,
    COUNT(DISTINCT c.id) as comment_count,
    GROUP_CONCAT(DISTINCT pm.meta_value) as meta
FROM posts p
INNER JOIN (
    SELECT id FROM posts 
    WHERE status = 'published' 
    AND created_at > '2024-01-01'
) filtered ON p.id = filtered.id
LEFT JOIN comments c ON p.id = c.post_id
LEFT JOIN post_meta pm ON p.id = pm.post_id
GROUP BY p.id
