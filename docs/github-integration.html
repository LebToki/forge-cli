GitHub Integration

Connect FORGE to GitHub for seamless repository management, PR creation, and AI-powered automation.

Authentication

1

Generate a GitHub Token

Go to GitHub Settings ‚Üí Developer settings ‚Üí Personal access tokens ‚Üí Tokens (classic)

Click "Generate new token" and select these scopes:

  • repo - Full control of private repositories
  • workflow - Update GitHub Action workflows
  • write:packages - Publish packages
  • delete:packages - Delete packages
2

Authenticate with FORGE

forge github auth

You'll be prompted to enter your token. It will be stored securely in your system keychain.

3

Verify Authentication

forge github list

This should display your repositories if authentication was successful.

üîê Security: Your token is never shared and is encrypted in your system's keychain. FORGE never sends your token to any server except GitHub's API.

AI-Powered Automation

ü§ñ Smart Workflows

Fix Issue and Create PR
forge github fix-issue owner/repo-name 123

FORGE will:

  1. Analyze the issue description
  2. Find relevant code files
  3. Generate a fix using AI
  4. Create a branch with the fix
  5. Commit changes
  6. Open a PR with description
  7. Link PR to the issue
Review PR with AI
forge github review-pr owner/repo-name 42

AI analyzes the PR and provides:

  • Code quality assessment
  • Potential bugs
  • Performance issues
  • Security concerns
  • Style violations
Generate Changelog
forge github changelog owner/repo-name --from v1.0.0 --to v1.1.0

AI creates a human-readable changelog from commits and PRs.
